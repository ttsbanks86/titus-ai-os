# PROMPT-MINE Cheat Sheet

## The 7 Patterns That Power Every Major AI Tool

### 1. IDENTITY LAYER
Every AI needs to know who it is.
```
"You are [NAME], created by [COMPANY]. Current date: [DATE]."
```
**→ Tell your AI who it is before you tell it what to do.**

### 2. AGENT LOOP
The most successful AI doesn't answer once — it loops.
```
ANALYZE → PLAN → EXECUTE → OBSERVE → ITERATE
```
**→ Think in cycles, not one-shots.**

### 3. ANTI-LIST BIAS
AI loves bullet points. Great AI fights that instinct.
```
"Avoid over-formatting with bold, headers, lists."
```
**→ Natural prose > bullet machines.**

### 4. CONSTRUCTIVE REFUSAL
Don't just say no — redirect.
```
"I can't do X, but I can help you with Y instead."
```
**→ Always offer an alternative.**

### 5. MEMORY RULES
Remember with consent, structure, and selectivity.
```
1. Only save what user volunteers
2. Tag with categories + timestamps
3. Load only what's relevant per session
```
**→ Smart memory > total recall.**

### 6. TOOL SIGNATURES
Treat tools like API functions, not loose descriptions.
```
tool_name(param: type) -> result
```
**→ Precision over prose.**

### 7. CORRECTION LAYER
Override the model's bad habits on purpose.
```
Explicit rules for what NOT to do.
```
**→ Your AI needs fences, not just roads.**
