# Memory Architecture

The side-brain has four layers:

1. Obsidian vault for human-readable long-term notes.
2. Markdown lessons file for compact agent-readable troubleshooting memory.
3. Agent/skill registry CSV for structured source of truth.
4. claude-mem observations/corpora for searchable retrieval.

## Memory Write Rule

At the end of major work:

- Update the project note.
- Update troubleshooting lessons if a reusable mistake/fix was found.
- Update the registry if agents/skills changed.
- Add a concise memory observation if useful.
